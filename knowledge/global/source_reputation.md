# Source Reputation

This file stores user-curated source reputation notes.

## Defaults

- Official product, documentation, regulatory, academic, and primary financial sources start with higher trust.
- Vendor blogs are useful for positioning but should not be treated as independent validation.
- SEO content, thin reposts, and anonymous summaries start with lower trust.
